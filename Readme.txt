Python version: 2.7

to install the packages:
	$ pip install -r requirements.txt

to run:
	$ python test.py <image_1> <image_2>

our threshold is 0.3
the intermediate output images are stored in output_images

Images of our database are stored in the folder my_db.
We have also included noisy images of a fingerprint. Fingerprint my_db/1_1.jpg is the original fingerprint and other images starting with name 1 (except 1_2.jpg) are noisy images. The type of noise will be clear by the name of the image
